## Naming Convention

Format: `YYYYMMDD_YuiQuery_Feature-Description.png`

- **Date prefix**: When screenshot was captured
- **Product identifier**: `YuiQuery_`
- **Feature description**: What the image shows

## Usage in Documents

Reference images in paper.md using relative paths:
```markdown
![Description](images/filename.png)
```

## Image Guidelines

- **Format**: PNG preferred for screenshots
- **Resolution**: Minimum 1200px width for clarity
- **Annotations**: Add callouts directly in image when helpful
- **Alt text**: Always include descriptive alt text in markdown
