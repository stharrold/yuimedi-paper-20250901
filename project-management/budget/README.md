# Budget Documentation

This directory contains budget planning and tracking documents for the YuiQuery Healthcare Analytics research project.

## Contents

| File | Description |
|------|-------------|
| `budget-final.json` | Final approved budget with detailed breakdown |

## Budget Structure

The budget includes:

1. **Primary Work** - Developer hours for 3 research papers
2. **Backup Coverage** - Risk mitigation for single point of failure
3. **SME Reviews** - Subject Matter Expert review hours
4. **Publication Fees** - Journal submission and conference costs
5. **Tools & Infrastructure** - Software and cloud services

## Budget Versions

- **v1.0**: Initial SME-only budget ($2,700)
- **v2.0**: Expanded with development costs
- **v3.0-REMEDIATED**: Final version with full scope ($33,495)
