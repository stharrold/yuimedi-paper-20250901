# IT Analytics Turnover Sources

**GitHub Issue:** #275
**Status:** Planning Complete

## Overview

This feature adds academic citations for IT/analytics staff turnover in healthcare, addressing a gap where the paper claims data about "clinical and technical staff" but only cites nurse-specific sources.

## Key Research Findings

- Healthcare IT staff: **2.9 years** average tenure, **15.54%** annual turnover
- This is the **highest turnover** among all IT organization types studied
- Hospital IT personnel: **4-6 years** average tenure

## New Citations

1. **Ang & Slaughter (2004)** - ACM SIGMIS Database
2. **Das et al. (2010)** - IEEE Transactions on Engineering Management
3. **Rodriguez (2020)** - ProQuest dissertation
4. **Belcher (2023)** - ProQuest dissertation
